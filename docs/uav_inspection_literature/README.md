# 无人机巡检论文与文献包

本目录收集了无人机巡检相关的开放获取论文、官方报告和技术指南，共 22 份 PDF。主题覆盖电力线路/绝缘子、桥梁与公路、光伏电站、风电设备、航线规划、自主飞行控制、视觉检测与数据集基准。

## 目录结构

- `pdfs/`: 已下载 PDF 文件。
- `sources/pdf_metadata.csv`: 自动读取的 PDF 页数、大小和元数据校验结果。
- `literature_index.csv`: 人工整理的文献索引、来源链接、DOI/编号、用途建议。
- `无人机巡检文献分析.md`: 中文分析报告和推荐阅读顺序。

## 可靠性筛选原则

1. 优先使用开放获取期刊、政府/交通主管部门报告、大学或研究团队公开论文、arXiv 预印本和正式 DOI 页面。
2. 对工程应用类论文单独标注为辅助材料，不作为核心理论依据。
3. 下载后使用 PDF 文件头和 `pypdf` 页数读取进行校验，避免把网页错误页当成论文。
4. 不使用绕过版权的下载源；不可合法公开下载的资料只记录来源，不纳入 PDF 包。

## 快速阅读建议

1. 先看总体综述：`P01`、`P02`、`B02`、`S01`、`W02`。
2. 做电力/YOLO 巡检项目：重点看 `P05`、`P06`、`P07`、`P08`、`P09`。
3. 做桥梁巡检方案或标准化流程：重点看 `B01`、`B02`、`B05`。
4. 做光伏热红外巡检：重点看 `S01`、`S02`、`S03`。
5. 做自主飞行与航线规划：重点看 `P03`、`B06`。
